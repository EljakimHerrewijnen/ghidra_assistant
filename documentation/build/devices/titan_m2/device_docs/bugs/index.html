fscryptctl

To build for ARM64

CC=$TOOLCHAINENV/android-ndk-r21_Linux/toolchains/llvm/prebuilt/linux-x86_64/bin/armv7a-linux-androideabi29-clang++ make

bluejay:/data/local/tmp # ./fscryptctl_arm64.elf get_policy /data/data
Encryption policy for /data/data:
        Policy version: 2
        Master key identifier: 056dbf8d2e9936b9b2f7462fda761a23
        Contents encryption mode: AES-256-XTS
        Filenames encryption mode: AES-256-CTS
        Flags: PAD_32, IV_INO_LBLK_64

generate dummy key

fout = open("/tmp/nfiekey", "wb")
fout.write(b"nfie" * 16) #512 bit key
fout.close()

Copy fscryptctl and key to /data/local/tmp and on the device inject the key:

$ su
|bluejay:/data/local/tmp # ./fscryptctl_arm64.elf add_key /data < /data/local/tmp/nfiekey
